# First Time Setup
* conda create --prefix ./env python=3.10
* conda activate ./env
* pip install -r requirements.txt
* Download `data` folder from https://drive.google.com/drive/folders/1RuK8OHl_pZzJavkxdgasE3os0dFnMGCp?usp=sharing, extract and put in project directory.
* streamlit run app.py

# For Subsequent Development
* conda activate ./env
* streamlit run app.py
